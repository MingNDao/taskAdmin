# 目标
个人任务管理，日历提醒每天需要完成的事情

# 工具准备
- 全局工具：node,npm,webpack
- 前端依赖：vue,vuex,vue-router,vue-resource,element-ui
- 后台依赖：express,path,body-parse
- 编译环境：gulp,browserify,babelify,vueify,babel-preset-es2015,gulp-sass

# 代码规范
- 代码严格按照 ESLint 默认规范

# 运行
``` bash
$ npm install
$ npm run test
```

# 修改编译
``` 
$ gulp
```
